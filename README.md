# 🚀 Smart Job Portal

A modern, full-stack job portal application designed to connect candidates with recruiters efficiently. Features a clean, responsive UI, role-based access control, and dynamic job listings.

## 🌟 Key Features
* **Role-Based Dashboards:** Distinct interfaces for Candidates, Recruiters, and Admins.
* **Advanced Job Search:** Clean UI to filter jobs by title, location, and keywords.
* **Secure Authentication:** UI integrated for JWT-based login and registration.
* **Responsive Design:** Fully responsive and modern layout built with Tailwind CSS.
* **Smart UI Components:** Interactive job cards, password visibility toggles, and seamless navigation.

## 💻 Tech Stack
* **Frontend Framework:** React.js (Vite)
* **Styling:** Tailwind CSS
* **Routing:** React Router DOM
* **Icons & UI:** Lucide React, React Icons
* **Alerts:** React Hot Toast
* **Backend Integration:** Axios (Connects to Spring Boot REST APIs)
